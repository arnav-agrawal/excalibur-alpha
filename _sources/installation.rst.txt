Installation
------------

As a user, you only have to follow 3 steps, all of which can be done in the terminal.

1. git clone https://github.com/arnav-agrawal/excalibur-alpha
2. cd excalibur-alpha
3. python setup.py install

Congratulations, you have installed Excalibur! You will now be able to run Excalibur on your local computer.

For examples showing how to use Excalibur after the installation step, look at the 'Getting Started' page.
